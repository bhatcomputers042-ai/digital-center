# 📷 Bhat Computers – Professional Photo Resizer

A modern, fast, and responsive **Photo Resizer** built with **HTML, CSS, and JavaScript**. It works entirely in the browser without requiring any server or internet connection.

---

## Features

* 📷 Upload images from your device
* 🖱️ Drag and drop image support
* 👀 Live image preview
* 📏 Resize by custom width and height
* 📊 Resize by percentage
* 🔒 Lock/unlock aspect ratio
* 🎨 Export as JPEG, PNG, or WebP
* 🎚️ Adjustable image quality
* 📁 Display original and resized dimensions
* 💾 Display original and estimated resized file size
* ⬇️ Download resized image
* 🔄 Reset tool with one click
* 📱 Responsive design for desktop, tablet, and mobile
* 🌐 Works completely offline
* ⚡ No external libraries required

## Installation

1. Download or clone the project.
2. Extract the files if downloaded as a ZIP.
3. Open `index.html` in any modern web browser.

No installation or server setup is required.

---

## How to Use

1. Open the application.
2. Upload an image or drag and drop it into the upload area.
3. Enter the desired width and height, or choose a resize percentage.
4. Select the output format (JPEG, PNG, or WebP).
5. Adjust the quality slider if needed.
6. Click **Resize Image**.
7. Preview the resized image.
8. Click **Download Image** to save it.
